# AI PPT 生成器 - HTML 框架

🎨 一个基于 HTML 的 AI PPT 生成器，能够生成演示文稿网页并分割成多页，支持导出 PPTX 框架数据。

## ✨ 功能特性

- 🚀 **纯 HTML 实现**：无需安装，浏览器直接打开即可使用
- 🤖 **AI 提示词生成**：自动生成适用于 AI 的完整提示词
- 📄 **多页分割**：支持生成 3-10 页的演示文稿
- 🎭 **多种风格**：专业商务、创意设计、学术研究、简约风格
- ✏️ **在线编辑**：可编辑和删除任意幻灯片
- 💾 **导出功能**：导出 PPTX 框架数据（JSON 格式）
- 📋 **提示词复制**：一键复制 AI 提示词
- 📱 **响应式设计**：支持桌面和移动设备

## 🚀 快速开始

### 方法 1：直接使用

1. 克隆或下载本仓库
2. 在浏览器中打开 `index.html` 文件
3. 开始使用！

### 方法 2：在线使用

直接访问 GitHub Pages 或将 `index.html` 部署到任何静态网站托管服务。

## 📖 使用说明

### 基本流程

1. **输入参数**
   - 填写演示文稿主题
   - 选择幻灯片数量（3-10页）
   - 选择演示风格
   - 添加补充说明（可选）

2. **生成幻灯片**
   - 点击"生成幻灯片"按钮
   - 系统将自动生成 AI 提示词和幻灯片预览

3. **编辑和调整**
   - 可以编辑任意幻灯片的标题
   - 可以删除不需要的幻灯片
   - 可以清空所有内容重新开始

4. **导出和使用**
   - 复制 AI 提示词，用于真实的 AI 生成
   - 导出 PPTX 框架数据，配合其他工具生成最终文件

## 📋 导出格式

导出的 JSON 文件包含以下信息：

```json
{
  "title": "演示文稿主题",
  "slideCount": 5,
  "style": "professional",
  "slides": [
    {
      "slideNumber": 1,
      "title": "标题",
      "content": "内容"
    }
  ],
  "metadata": {
    "generated": "2024-01-14T00:00:00.000Z",
    "prompt": "AI提示词"
  }
}
```
